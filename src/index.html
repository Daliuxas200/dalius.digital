    Atgal į Gamtą

    • About

    • Stack

    Atgal į Gamtą is wildlife rehabilitation Non-profit organisation based in Vilnius, Lithuania. Their focus is education of the public, rehabilitation of small birds and mammals, and their eventual reintroduction into the wild. Operation is almost entirely volunteer based and funded by donations, thus and having an engaging Web presence was paramount to grow the operations.

    Follow the link to their website to learn more about their great work!

    • Wordpress powered custom theme, professionaly designed by Laura Diržytė
    • Fully customizeable Homepage with reusable and fully customizable Drag and Drop sections
    • Ability to include any custom Section into any other Page or Post
    • Effective use of CSS and JS preprocessors, image compression and Cache control, resulting in a lightweight Website for fast serving speeds.
    • Custom editor Blocks

    Oor.lu

    • About
    • Features
    • Stack

    Oor.lu is a lightweight url shortener. Utilising minimalistic design phylosophy and requiring zero dependencies to run the Front-end, resulted in an ultra light weight website for lightning fast presentation. Backend created by Adam Wojtczak using django-REST (Python), PostgreSQL and Redis cache. Design by Austėja Slavickaitė Wojtczak.

    Visit the site to try it out and learn more!

    • Webpack powered SCSS, JS and image preprocessing for ultra-lightweight package and serving speeds.
    • Vanilla JS written interactive front end, requiring zero dependencies.
    • Professional Design by Austėja Slavickaitė Wojtczak
    • Professional Backend by Adam Wojtczak
